使用方法

每天1次，每次1袋。洁面后可以全脸先涂抹少量艺霏虾青素莹润修护精华液，再贴敷艺霏巴布贴，贴敷1-2小时，连续使用不少于8周。 揭除敷贴后不用清洗，再次涂抹艺霏虾青素莹润修护精华液，修复皮肤屏障功能，加速精华液中的神经酰胺、超低分子量透明质酸等活 性成分的透皮吸收。根据使用者皮肤敏感程度决定最佳贴敷时间：皮肤很不敏感的，可以延长贴敷时间，甚至可贴敷过夜。敏感肌肤者 在脸颊部位的贴敷时间不要超过1小时，随后将贴片移至其它部位继续贴敷。

  • 虾青素来自夏威夷纯天然雨生红球藻，是自然界最强的抗氧化剂，清除自由基的能力是VC的6000倍、番茄红素的1800倍、VE的1000倍、花青素的700倍、茶多酚的320倍、胡萝卜素的100倍。

  • 精华液中含有高浓度的神经酰胺（糖鞘脂类），可以补充角质层缺失的神经酰胺，修补漏洞，减少水分丢失，从而修护皮肤屏障功能。精华液中还添加有高浓度的超低分子量透明质酸，一方面帮助皮肤锁住外部水分，同时透皮吸收进入皮肤，补充皮肤组织中缺失的透明质酸，有利于减少皱纹。产品中还含有皮脂膜固有成分角鲨烯，也能补充一部分脂质，修补屏障。

  • 产品中的葡聚糖具有修护皮肤损伤、减少炎症反应等多种功能，还能促进角质细胞的生长，持续使用一段时间后，角质层层数增多，慢慢接近正常状态。
